== Description ==

Shortcodes Everywhere is a plugin which allows you to edit your site from the front-end, at any part you want.

Just activate the plugin, and than from the front-end press " **'** " (it is the character just down to Escape with a nibble ~) on your keyboard to enable debug mode, hover through areas you need to edit and click on them.

It accepts HTML and Shortcodes as well.

If you need to remove any of the modifications you've done, than go to your Admin Dashboard, and edit or delete Live Edits.

On those Custom Posts, you will also be able to edit the content, or change the CSS Selector where the modification is being injected.

== Installation ==

1. Upload `shortcode-everywhere.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Login as Admin, and view the site from front-end
4. Press "'" on your keyboard, and select the area you need to edit
